Install git globally

npm i git -g


1. in the root folder of your project create a file named .gitignore

its content for start is those lines

node_modules

config/default.json

which mean "don't include the node_modules folder and the config/default.json file

2. In the console type git init  // initiates a repo

3. git add .  // add all files in order to commit

4. git commit -m 'YOUR_DESCRIPTIVE_MESSAGE'  // make the commit

YOUR_DESCRIPTIVE_MESSAGE is just some text


5. Make  a github account and login to github

6. From the top-right select the + sign , New Repository

7. Set a name, e.g myrepo

After on, if you update your code you can be updating your local commits with

git add .

git commit -m 'YOUR_NEW_COMMIT_MESSAGE"

and also update the github remote copy with

git push
